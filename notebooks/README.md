# 分析笔记本目录

本目录包含用于数据分析和实验的Jupyter笔记本文件。

## 笔记本说明

### 数据分析
- `env_params_eda.ipynb` - 环境参数探索性数据分析
- `clip_test.ipynb` - CLIP模型测试和验证

## 使用方法

```bash
# 启动Jupyter Lab
jupyter lab

# 或启动Jupyter Notebook
jupyter notebook
```

## 开发规范

1. 笔记本应该有清晰的标题和说明
2. 代码单元应该有适当的注释
3. 分析结果应该有文字说明
4. 重要的分析应该保存为脚本形式
5. 笔记本输出应该清理后再提交

## 注意事项

- 笔记本主要用于探索性分析和实验
- 生产代码应该放在src目录下
- 大型数据文件不应该保存在笔记本中
- 敏感信息不应该出现在笔记本中
