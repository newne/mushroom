# 测试目录

本目录包含蘑菇图像处理系统的测试文件。

## 测试文件说明

### 功能测试
- `test_cache_optimization.py` - 缓存优化功能测试

## 运行测试

```bash
# 运行所有测试
python -m pytest tests/

# 运行特定测试
python tests/test_cache_optimization.py
```

## 测试规范

1. 所有测试文件应以 `test_` 开头
2. 测试函数应以 `test_` 开头
3. 测试应该独立运行，不依赖外部状态
4. 重要的功能都应该有对应的测试用例

## 注意事项

- 临时测试文件不应提交到版本控制
- 测试数据应使用模拟数据，避免依赖真实环境
- 测试应该快速执行，避免长时间运行的集成测试
